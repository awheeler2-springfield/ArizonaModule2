Module 2 — Source Intake

After deciding that a claim needs evidence, a healthcare writer must identify what kind of source is being used. Source types do different work. A writer should be able to classify a source and judge how that source should be used in healthcare writing.

In this module, you will review source situations and decide what kind of source is being used, what function it serves, and how appropriate it is for the writing task.

Key Principle

Source intake is not just naming a source type. It is deciding what kind of source this is, what it does, and how it should be used.

In healthcare writing, different sources serve different functions: background explanation, original evidence, public guidance, professional recommendation, or current context.

When you are ready, begin the source intake review and evaluate each source situation.

Source Intake Note

Explain how identifying the type of source affects how it should be used in healthcare writing.

Documentation Note
Model Documentation Statement

Identifying the type of source matters because different sources do different kinds of work in healthcare writing. A scholarly article may provide original research evidence, a clinical guideline may offer evidence-based recommendations, and a healthcare organization publication may summarize trusted public health guidance. Reference sources are useful for background explanation, while news articles are better for recent context than for primary evidence. Classifying the source helps the writer decide how strong the evidence is, how appropriate it is for the claim, and what limitations it may have.
